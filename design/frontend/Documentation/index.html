B) CSS Files and Structure - top

There are three css files necessary for customization.

  1. "css/main.css" -Is the main css file.
  2. "css/responsive.css" -Is the responsive css file.
  3. "css/elements.css" -Is the elements css file ( Check elemets.html page for ease of using the elements ).
  4. "css/ie.css" -Is the IE 8+ Fallback css file.
  5. "css/calendar.css" -Is the calendar css file.

The main.css file is separated sections:

C) JavaScript and jQuery Libraries - top

  1. "js/main.js" -Has all the important jQuery calls
  2. "js/bootstrap.js" -Twitter Bootstrap framework
  3. "js/vendor/jquery-1.10.1.min.js" -jQuery JavaScript Library
  4. "js/calendar.js" -Calendar JS
  5. "js/jflickrfeed.min.js" -Flicker Widget JS
  6. "js/jquery.customSelect.min.js" -Styled selector among browsers JS
  7. "js/jquery.flexslider-min.js" -for all the sliders
  8. "js/jquery.smartmenus.min.js" -For the responsive menu
  9. "js/newsTicker.js" -For the Breaking News Ticker
  10. "js/placeholders.min.js" -View HTML5 Placeholder for old browsers
  11. "js/retina-1.1.0.min.js" -View High quality images on Retina / High Res screens
